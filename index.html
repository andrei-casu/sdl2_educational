Noțiuni Introductive
Pixel. Ce este un pixel?

Un pixel este un element component foarte mic al unei imagini digitale.
Pe scurt, este un punct care are o culoare.


Pixelii unui monitor:

Un monitor are o rezoluție care se masoară in pixeli.
Un exemplu este 1920x1080.
- 1920 reprezintă numărul de pixeli care încap pe lungime
- 1080 reprezintă numărul de pixeli care încap pe lățime.

În funcție de numărul de pixeli al unui monitor,
acesta poate să arate mai multe/mai putine detalii.

Sistem de coordonate

- intr-un sistem de coordonate cum se învață la mate,
(primul sistem de coordonate) punctul (0,0) se află În
colțul stânga jos

- în grafica pe calculator punctul (0,0) se află
în colțul stânga sus.

INITGRAPH

Care este structura?

void initgraph(int *gd, int *gm, char *pathtodriver);


Care sunt parametrii?

funcția are 3 parametrii:

- gd = graphics driver
- gm = graphics mode
- pathtodriver = adresa folderului în care se află fișierele BGI.

DETECT - macro definit in "graphics.h"
valoarea numerică = 0(requests autodetect)

în cazul in care gd primeste valoarea
DETECT gm este setat cu ajutorul funcției
la cea mai înaltă rezoluție disponibilă
pe calculatorul/laptopul respectiv

Codul în care se apelează
 #include <graphics.h>
 #include <iostream>
 using namespace std;


 int main()
 {
	int gd = DETECT, gm;

	initgraph(&gd, &gm, "C:\\TC\\BGI");inițializează modul graphics

  	getch();așteaptă până a fost apăsată o tastă
  	closegraph();închide modul grafic
  	return 0;
 }


Cum se apeleaza?

initgraph(&gd, &gm, "C:\\TC\\BGI");

Ce face?

- inițializează modul graphics

CLOSEGRAPH

Care este structura?

void closegraph(int wid=ALL_WINDOWS);


Care sunt parametrii?

funcția are 1 parametru OPȚIONAL:

- wid = ce anume se închide

acesta poate avea următoarele valori:
ALL_WINDOWS - închide toate ferestrele
CURRENT_WINDOW - închide doar fereastra curentă

Codul în care se apelează
 #include <graphics.h>
 #include <iostream>
 using namespace std;
 
 int main()
 {
    int gd = DETECT, gm;
 
    initgraph(&gd, &gm, "C:\\TC\\BGI");inițializează modul graphics
 
    outtext("Press any key to close the graphics mode...");afișează textul
 
    getch();așteaptă până a fost apăsată o tastă
    closegraph();
    return 0;
 }


Cum se apeleaza?

closegraph();

Ce face?

- închide modul grafic
- eliberează toată memoria alocată
- aduce fereastra în stadul inițial

GRAPHDEFAULTS

Care este structura?

void graphdefaults(void);


Care sunt parametrii?

funcția nu are parametrii

Codul în care se apelează
 #include <graphics.h>
 #include <iostream>
 using namespace std;

 int main()
 {
   int gd = DETECT, gm;
 
   initgraph(&gd, &gm, "C:\\TC\\BGI");inițializează modul graphics
 
   setcolor(RED);setează culoarea
   setbkcolor(YELLOW);setează culoarea de fundal
 
   circle(250, 250, 50);desenează un cerc
 
   getch();așteaptă până a fost apăsată prima tastă
   graphdefaults();
 
   getch();așteaptă până a fost apăsată a doua tastă
   closegraph();închide modul grafic
   return 0;
 }


Cum se apeleaza?

graphdefaults();

Ce face?

- setează fereasta la dimensiunile ecranului
- mută poziția curentă la coordonatele 0 0
- resetează culorile, stilurile și fonturile

ARC

Care este structura?

void arc(int x, int y, int stangle, int endangle, int radius);


Care sunt parametrii?

funcția are 5 parametrii:

- x = coordonata x
- y = coordonata y
- stangle = la cate grade incepe
- endangle = la cate grade se termina
- radius = raza cercului din care face parte

gradele sunt poziționate în felul urmator:
0 grade - ora 3
90 grade - ora 12
180 grade - ora 9
270 grade - ora 6

Codul în care se apelează
 #include <graphics.h>
 #include <iostream>
 using namespace std;

 int main()
 {
   int gd = DETECT, gm;
 
   initgraph(&gd, &gm, "C:\\TC\\BGI");inițializează modul graphics
 
   arc(100, 100, 0, 135, 50);
 
   getch();așteaptă până a fost apăsată o tastă
   closegraph();închide modul grafic
   return 0;
 }


Cum se apeleaza?

arc(100, 100, 0, 135, 50);

Ce face?

- desenează arcul de cerc cu dimensiunile specificate

LINE

RECTANGLE

Care este structura?

void drawpoly(int numpoints, int *polypoints);


Care sunt parametrii?

funcția are 2 parametrii:

- numpoints = n (numărul de vârfuri)
- polypoints = lista coordonatelor vârfurilor

numpoints are valoarea numărului de vârfuri

dacă vrem ca poligonul să fie închis primul varf
trebuie specificat de 2 ori
prin urmare lista de vârfuri trebuie să înceapă și
să se termine cu aceleași coordonate (x, y)


Codul în care se apelează
 #include <graphics.h>
 #include <iostream>
 using namespace std;

 int main()
 {
   int gd = DETECT, gm;
   int points[]={320, 150, 420, 300, 250,300, 320, 150};

   initgraph(&gd, &gm, "C:\\TC\\BGI");inițializează modul graphics
    
   drawpoly(4, points);
 
   getch();așteaptă până a fost apăsată o tastă
   closegraph();închide modul grafic
   return 0;
 }


Cum se apeleaza?

drawpoly(4, points);

Ce face?

- desenează un poligon cu numpoints vârfuri folosind culoarea curentă
- ia punctele din lista dată 2 câte 2 pentru coordonatele vârfurilor
