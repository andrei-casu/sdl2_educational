INITGRAPH

1. care este structura?
void initgraph(int *gd, int *gm, char *pathtodriver);

2. care sunt parametrii?

functia are 3 parametrii:

gd = graphics driver
gm = graphics mode

*DETECT - macro definit in "graphics.h"
valoarea numerica = 0(requests autodetect)

in cazul in care gd primeste valoarea
DETECT gm este setat cu ajutorul functiei
la cea mai inalta rezolutie disponibila
pe calculatorul/laptopul respectiv

pathtodriver = adresa folderului in care se afla
fisierele BGI.

3. cum se apeleaza?

initgraph(&gd, &gm, "C:\\TC\\BGI");

4. codul in care se apeleaza

#include
#include

int main()
{
int gd = DETECT, gm;

initgraph(&gd, &gm, "C:\\TC\\BGI");

getch();
closegraph();
return 0;
}

5. ce face?
initgraph - initializeaza modul graphics
getch - asteapta pana a fost apasata o tasta
closegraph - inchide modul grafic
