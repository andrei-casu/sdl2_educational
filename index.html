INITGRAPH

Care este structura?

void initgraph(int *gd, int *gm, char *pathtodriver);

Care sunt parametrii?

funcția are 3 parametrii:

->gd = graphics driver
->gm = graphics mode
->pathtodriver = adresa folderului în care se află
fișierele BGI.

*DETECT - macro definit in "graphics.h"
valoarea numerică = 0(requests autodetect)

în cazul in care gd primeste valoarea
DETECT gm este setat cu ajutorul funcției
la cea mai înaltă rezoluție disponibilă
pe calculatorul/laptopul respectiv

Codul în care se apelează:
 
 #include<graphics.h>
 #include<conio.h>

 int main()
 {
	int gd = DETECT, gm;

	initgraph(&gd, &gm, "C:\\TC\\BGI");inițializează modul graphics

  	getch();așteaptă până a fost apăsată o tastă
  	closegraph();închide modul grafic
  	return 0;
 }
